### [정렬]

### K번째수 *

**JS의 `정렬기준은 문자열`로, arr.sort() 시 arr = [100, 2] 일 때, "100" < "2"로 인식해 정렬한다.**

**따라서 정수형으로 정렬을 위해서는 다음과 같이 코드를 작성해야 한다.**

```javascript
arr.sort((a,b)=> {
  return a - b;
})
```

### 타겟 넘버
